⚙️ Setup Google Sheets

How to get your credentials:
  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable "Google Sheets API"
  4. Create credentials (API Key) and copy it
  5. In your Google Sheet, click Share → Copy the Sheet ID from URL
Sheet ID is the long string between /d/ and /edit in your sheet URL
Example: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms
Make sure your columns are: SKU, PRODUCT Name, Quantity
